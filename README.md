# Happenings — Brochure Site

Static single-page site (no build step). `index.html` contains everything (HTML/CSS/JS inline).

## Local preview
Just open `index.html` in a browser, or:
```
npx serve .
```

## Deploy
Push this repo to GitHub, then import it in Vercel as a static project (no framework, no build command, output directory = `/`).
